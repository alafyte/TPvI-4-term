ДОСЛОВНОЕ УСЛОВИЕ ИЗ БИЛЕТА

Используя драйвер JDBC выполнить организацию соединения с БД.
Она должна содержать author, book_name, publication_year, pages.
Написать страницу JSP с кнопкой “NEXT”, по нажатию на которую
будет выводиться последовательно по две записи( в виде таблицы с заголовками) из бд.
Используйте maven для сборки.
